Pathfinder Engine

03

A graph-based pathfinding visualizer that implements and benchmarks multiple search algorithms on dynamic grids. Users can place obstacles, set start and end points, and watch algorithms solve the shortest-path problem in real time.

C++ Algorithms Graph Theory Data Structures
Problem Understanding the tradeoffs between search algorithms — BFS, Dijkstra's, A* — requires seeing them operate on identical inputs and comparing their behavior and performance.
What I Built An interactive engine that generates grids, runs multiple pathfinding algorithms, and outputs comparative metrics including nodes explored, path length, and execution time.

Cache Simulator

04

A configurable CPU cache simulator that models direct-mapped, set-associative, and fully-associative caches. The tool processes memory access traces and reports hit rates, miss rates, and eviction patterns across different cache configurations.

C Systems Programming Computer Architecture Performance Analysis
Problem Cache design decisions directly impact system performance. Comparing configurations requires a tool that accurately simulates memory hierarchy behavior under realistic workloads.
What I Built A command-line simulator that accepts configurable parameters — cache size, block size, associativity, replacement policy — and processes trace files to output detailed performance statistics.
